# Slash Notation
- More concise way of showing a [[Subnet Mask]]
- Is the number of 1s after converting the subnet mask into [[Binary]]

```
192.168.1.0/24

255     .255     .255     .0        
11111111.11111111.11111111.00000000 = 24 1s
```
